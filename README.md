Qt-Code-Editor
==============

Codeeditor mit Netzwerkfunktion

  * Code Completion
  * Syntax Highlighting
  * File Browser
  * Document Browser
  * Network Data Sharing
  * Network WorkChatting
  * MDI
  * Docks
  * Current Line Highlighting
  * Line Numbers
  * Search & Replace
  * Print (PDF)
  * Recent Files
  * Toolbar
  * Settings
  * Statusbar
  * Adaptiv Coloring
